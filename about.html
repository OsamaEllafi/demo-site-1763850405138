About Us

Welcome to Amazon Mockup, a frontend project designed to replicate the key visual and functional aspects of the Amazon user experience. This project showcases responsive design, interactive elements, and modern web technologies to mimic Amazon's clean, utilitarian e-commerce interface.

Built with HTML, CSS, and JavaScript, it includes features like product carousels, shopping cart functionality using localStorage, image galleries, and fully responsive layouts for desktop, tablet, and mobile devices.
